# 🌐 Vansh Network Analyzer

A modern Network Analyzer Dashboard built using HTML, CSS and JavaScript.

This project demonstrates networking concepts through an interactive dashboard with a modern cyber security inspired interface.

# 📖 About Project

Vansh Network Analyzer is a frontend web application that simulates common networking tools such as IP Lookup, DNS Lookup, Ping Test, Port Scanner, WHOIS Lookup, and Traceroute. It also displays live network statistics using interactive charts.

This project is built for educational purposes and showcases frontend development skills.

### 🌍 DNS Lookup

Shows

- Response Time
- Latency
